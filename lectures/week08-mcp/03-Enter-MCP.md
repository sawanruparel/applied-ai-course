# Model Context Protocol

An open standard for agents to interact with tools, data, and environments.

Released by Anthropic on November 25, 2024. Inspired by LSP (Language Server Protocol) — the same idea that standardized IDE-to-language-server communication, applied to AI-to-tools.

Now adopted by Claude, ChatGPT, VS Code Copilot, Cursor, Zed, and more.

## LLMs Today
- No tool access by default — every integration is hand-built
- No persistent memory across sessions
- No standard way to connect to databases, APIs, or filesystems

## What Happens
- Teams embed tool schemas directly in system prompts
- Every company writes its own connector code
- Switching models means rewriting integrations
- OpenAI, Google, Anthropic each have different function-calling formats

> Every team rebuilds the same infrastructure — there are 10+ ways to give an LLM a "search" tool, none interoperable.

# MCP Research Landscape (2025–2026)

- 10+ academic papers published since MCP's late-2024 launch
- Majority focus on **security** — the protocol's biggest open question
- Key venues: arXiv, OpenReview, Preprints.org

| Paper | Focus | So What? |
|-------|-------|----------|
| [Hou et al. — Landscape & Security Threats](https://arxiv.org/abs/2503.23278) | First comprehensive ecosystem analysis | Gives us the threat model — if you're deploying MCP, start here |
| [CA-MCP (Jayanti et al.)](https://arxiv.org/abs/2601.11595) | Context-aware multi-server collaboration | Shows how to cut LLM calls and failures in multi-server setups |
| [Breaking the Protocol (Maloyan & Namiot)](https://arxiv.org/abs/2601.17549) | Protocol-level vulnerability analysis | Proves MCP's trust model is too permissive — spec changes needed |
| [MCPShield (Zhou et al.)](https://arxiv.org/abs/2602.14281) | Adaptive trust calibration layer | First working defense layer you could actually deploy today |
| [SoK: Security & Safety (arXiv:2512.08290)](https://arxiv.org/abs/2512.08290) | Synthesizes 7 CVEs, 11 incidents | The field's "state of the union" — use it as your security checklist |

> **So what for builders:** this is a protocol where the security research is ahead of the security tooling. Read the papers before you ship.

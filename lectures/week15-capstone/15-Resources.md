# Resources for Continued Learning

### Frameworks and Tools
- **[LangChain](https://www.langchain.com/) / [LangGraph](https://langchain-ai.github.io/langgraph/)** — agent orchestration and graph-based workflows
- **[LlamaIndex](https://www.llamaindex.ai/)** — data connectors and retrieval pipelines
- **[MCP SDK](https://modelcontextprotocol.io)** — build and consume MCP servers ([TypeScript](https://github.com/modelcontextprotocol/typescript-sdk), [Python](https://github.com/modelcontextprotocol/python-sdk))
- **[vLLM](https://vllm.ai/) / [Ollama](https://ollama.com/)** — local model serving for development and testing
- **[Weights & Biases](https://wandb.ai/site/) / [MLflow](https://mlflow.org/)** — experiment tracking and model management

### Courses and Books
- *[Designing Machine Learning Systems](https://www.oreilly.com/library/view/designing-machine-learning/9781098107956/)* (Chip Huyen) — production ML fundamentals
- *[Building LLM Apps](https://github.com/anthropics/claude-cookbooks)* (Anthropic Cookbook) — practical patterns and recipes
- [Stanford CS224N](https://web.stanford.edu/class/cs224n/) — NLP foundations if you want deeper theory
- [Fast.ai](https://www.fast.ai/) — practical deep learning, updated regularly

### Key Papers by Course Module
- Prompt engineering: [Chain-of-Thought (Wei et al., 2022)](https://arxiv.org/abs/2201.11903), [ReAct (Yao et al., 2023)](https://arxiv.org/abs/2210.03629)
- RAG: [Self-RAG (Asai et al., 2023)](https://arxiv.org/abs/2310.11511), [RAPTOR (Sarthi et al., 2024)](https://arxiv.org/abs/2401.18059)
- Agents: [Toolformer (Schick et al., 2023)](https://arxiv.org/abs/2302.04761), [MCP specification (Anthropic, 2024)](https://modelcontextprotocol.io)
- Distillation: [DistilBERT (Sanh et al., 2019)](https://arxiv.org/abs/1910.01108), [knowledge distillation survey (Gou et al., 2021)](https://arxiv.org/abs/2006.05525)
- Safety: [Constitutional AI (Bai et al., 2022)](https://arxiv.org/abs/2212.08073), [red-teaming LLMs (Perez et al., 2022)](https://arxiv.org/abs/2202.03286)

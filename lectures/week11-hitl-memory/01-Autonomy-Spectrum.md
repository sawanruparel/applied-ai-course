# The Autonomy Spectrum

Where does your agent sit?

## Step 1: Full Manual

Human does everything. AI provides information only.

- Traditional search, dashboards
- Zero autonomy, maximum control
- Example: ChatGPT in Q&A mode

## Step 2: Copilot

AI suggests, human decides and executes.

- Code completion, writing assistants
- Human approves every action
- Example: GitHub Copilot suggesting code

## Step 3: Supervised Autonomy

AI acts, human oversees and intervenes when needed.

- Agent executes with approval gates
- Human reviews critical decisions
- Example: AI scheduling assistant that confirms before sending invites

## Step 4: Full Autonomy

AI acts independently within defined boundaries.

- Fully autonomous pipelines
- Human sets policy, agent executes
- Example: Automated trading within risk limits

---

**Key insight:** Most production agents should live in Steps 2-3. Full autonomy is rarely appropriate for high-stakes domains.

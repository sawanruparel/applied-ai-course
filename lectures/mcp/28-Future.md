# Future Direction

- **Streamable HTTP transport** — replacing SSE; enables resumable connections and server-initiated messages
- **OAuth 2.1 integration** — standardized auth flow for remote MCP servers (spec draft March 2025)
- **Elicitation** — servers can ask the user clarifying questions mid-execution
- **Agent-to-agent protocols** — complementary specs like A2A (Google) for cross-agent orchestration
- **Registry and discovery** — centralized server registries for easier capability lookup

# One Protocol, Many Ecosystems

A filesystem server built for Claude Desktop works unchanged in Cursor, Zed, Windsurf, or any MCP client. SDKs exist in TypeScript, Python, Java, Kotlin, C#, Swift, and Rust.

The same `tools/call` contract works whether the server wraps a local CLI, a cloud API, or a database.

> Interoperability is the moat — the more clients and servers adopt MCP, the more valuable each one becomes.

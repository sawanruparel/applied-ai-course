## HTTP
Standardizes how web services talk. Any client, any server, one protocol. Built on request/response over TCP.

## MCP
Standardizes how agents connect to capabilities. Any LLM, any tool server, one protocol. Built on JSON-RPC 2.0 over stdio or HTTP.

> The official analogy: MCP is "USB-C for AI applications" — one connector, many devices.

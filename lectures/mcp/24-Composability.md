# Add or Remove Capabilities

Need database access? Add `server-postgres`. Need it gone? Remove the config line. The agent code never changes.

Today: filesystem + GitHub. Tomorrow: filesystem + GitHub + Jira + Slack. The capability surface grows by configuration, not code.

> MCP makes agent capabilities a deployment concern, not a development one.

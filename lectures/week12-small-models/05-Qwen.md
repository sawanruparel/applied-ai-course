# Qwen 2.5 and Qwen 3

## Qwen 2.5 (Sept 2024) — The Quiet Powerhouse

- Full size range: **0.5B, 1.5B, 3B, 7B, 14B, 32B, 72B**
- Best-in-class multilingual support (29+ languages)
- Exceptionally strong on math (GSM8K, MATH) and coding (HumanEval)
- Qwen 2.5 7B rivals Llama 3.1 70B on several benchmarks
- Apache 2.0 license for models up to 72B

## Qwen 2.5 Coder

- Dedicated coding variants at 1.5B, 7B, 14B, 32B
- Qwen 2.5 Coder 32B matches GPT-4o on code generation tasks
- Supports 90+ programming languages

## Qwen 3 / 3.6 (2025–2026)

- Hybrid thinking models: can toggle between "thinking" and "non-thinking" modes
- Sizes from **0.6B to 235B** (MoE with 22B active)
- Qwen 3 8B matches Qwen 2.5 72B on reasoning benchmarks
- **Qwen 3.6 27B (May 2026)** hits **77.2% on SWE-bench**, a new high-water mark for an open small model
- 128K context with YaRN-based extension to 1M tokens
- Apache 2.0 license across all sizes

## Why Qwen Matters

- Strongest small models for non-English languages
- Math and code performance punches well above weight class
- Fully open weights — no usage restrictions

# The Best Model Is the One You Can Afford to Run

## At scale, model selection is an economic decision as much as a technical one.

---

### This Week's Assignment

1. Pick a task from your project (classification, extraction, generation, or Q&A)
2. Build a 200-example eval set from real or realistic data
3. Benchmark at least 3 models: one frontier, one mid-size, one small (<8B)
4. Plot the cost-quality Pareto frontier
5. Write a 1-page recommendation: which model would you deploy and why?

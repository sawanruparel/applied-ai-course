# The Cost Reality

## Frontier Model Pricing (per 1M tokens, as of early 2026)

- **GPT-4.1**: $2.00 input / $8.00 output
- **Claude Sonnet 4**: $3.00 input / $15.00 output
- **Claude Opus 4**: $15.00 input / $75.00 output
- **Gemini 2.5 Pro**: $1.25–$10.00 input / $2.50–$30.00 output
- **o3-pro**: $20.00 input / $80.00 output

## Small Model Pricing

- **GPT-4.1 mini**: $0.40 input / $1.60 output
- **GPT-4.1 nano**: $0.10 input / $0.40 output
- **Claude Haiku 3.5**: $0.80 input / $4.00 output
- **Gemini 2.0 Flash**: $0.10 input / $0.40 output
- **Self-hosted Llama 3.2 3B**: ~$0.02–0.05/M tokens (compute only)

## The Math

A pipeline processing 100M tokens/month:
- With Claude Opus 4: **~$4,500/month**
- With GPT-4.1 nano: **~$25/month**
- With self-hosted Llama 3.2 3B: **~$3/month**

That is a **180x–1,500x cost difference** for tasks where small models perform comparably.

# Multimodality: Not Just Text Anymore

## Vision

- All frontier models now accept images natively
- Chart/diagram understanding, OCR, screenshot analysis
- UI understanding enables computer use agents
- Medical imaging, satellite imagery, document processing

## Audio

- Speech-to-text and text-to-speech integrated into models
- Gemini: native audio understanding (tone, music, ambient sound)
- GPT-4o: real-time voice conversation mode
- Whisper/open alternatives for self-hosted speech processing

## Video

- Gemini 2.0: process hours of video directly
- Frame-by-frame analysis without external tools
- Surveillance, content moderation, sports analysis
- Still early: high cost, limited availability

## Code & Structured Data

- Models understand and generate code in 50+ languages
- JSON/XML/YAML structured outputs with schema validation
- SQL generation from natural language
- Direct interaction with databases and APIs

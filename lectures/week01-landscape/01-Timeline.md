# Key Milestones: 2023-2025

- **Mar 2023** -- GPT-4 launches: multimodal, passes the bar exam (90th percentile)
- **Nov 2023** -- GPT-4 Turbo: 128K context, cheaper API pricing
- **Dec 2023** -- Mixtral 8x7B: open-weight MoE proves efficient alternatives exist
- **Mar 2024** -- Claude 3 Opus: competitive with GPT-4, strong on analysis and safety
- **Apr 2024** -- Llama 3 (8B, 70B): Meta's open-weight push accelerates the ecosystem
- **Jun 2024** -- Claude 3.5 Sonnet: faster than Opus, matches or exceeds it on benchmarks
- **Sep 2024** -- OpenAI o1: first commercial "reasoning model" with chain-of-thought at inference
- **Jan 2025** -- DeepSeek R1: open-weight reasoning model trained with RL, rivaling o1
- **2025** -- o3, Claude 4, Gemini 2.0: reasoning and tool use become table stakes

> The pace of change is not slowing down -- it is accelerating and shifting direction.

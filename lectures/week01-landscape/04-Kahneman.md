# Kahneman's Framework Applied to LLMs

## System 1: Fast Thinking

- Automatic, effortless, intuitive
- Pattern matching from experience
- "What is 2 + 2?" -- instant
- Prone to cognitive biases
- **LLM analogy:** single forward pass, next-token prediction, pattern completion
- Good enough for most everyday tasks

## System 2: Slow Thinking

- Deliberate, effortful, logical
- Step-by-step reasoning
- "What is 347 x 283?" -- requires work
- Can override System 1 errors
- **LLM analogy:** chain-of-thought, self-verification, search over reasoning paths
- Essential for complex, novel problems

> Inference-time scaling gives LLMs their first real "System 2" -- the ability to think harder when the problem demands it.

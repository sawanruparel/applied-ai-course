# The Open-Weight Landscape

## Llama 3 (Meta)

- 8B, 70B, 405B parameter variants
- Permissive license for commercial use
- 405B competitive with GPT-4 on many tasks
- Massive ecosystem: fine-tunes, quantizations, tooling
- Foundation for thousands of derivative models

## Qwen 2.5 (Alibaba)

- 0.5B to 72B parameters
- Strong multilingual performance (especially CJK)
- Qwen2.5-Coder: specialized coding variants
- Competitive with Llama 3 at equivalent sizes
- Apache 2.0 license

## Mistral / Mixtral

- Mixtral 8x22B: MoE architecture, efficient inference
- Mistral Large: closed-weight flagship
- Strong European AI alternative
- Pioneered efficient MoE for open models

## Phi (Microsoft)

- Phi-3/Phi-4: "small language models" (3.8B-14B)
- Trained on high-quality synthetic + curated data
- Punches well above weight class on reasoning
- Ideal for edge deployment and fine-tuning

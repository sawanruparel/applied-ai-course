# Claude: From 3 to 4

### Claude 3 family (March 2024)
- Haiku (fast/cheap), Sonnet (balanced), Opus (best quality)
- First to seriously challenge GPT-4 across benchmarks
- Strong instruction following, safety, and long-context performance

### Claude 3.5 Sonnet (June 2024)
- Matched or exceeded Opus quality at Sonnet speed and cost
- SWE-bench Verified: 49% (state-of-the-art at the time)
- Introduced **computer use** (beta): model can operate a desktop GUI

### Claude 4 (2025)
- **Extended thinking:** explicit chain-of-thought reasoning (visible to developers)
- Up to 128K+ thinking tokens on hard problems
- Strong tool use: multi-step function calling with planning
- Computer use GA: autonomous browser and desktop interaction
- 200K context window standard

### Distinctive strengths:
- Careful instruction following and nuanced safety behavior
- Strong at long-document analysis and structured outputs
- Extended thinking is developer-controllable (budget tokens for reasoning)
